# Getting Help

## Documentation

Start with the [README](README.md) for installation, usage examples, and an explanation
of the Ornstein-Uhlenbeck model and parameter interpretation.

## Questions & Discussion

For usage questions, analytical questions about mean reversion / pairs trading, or
general discussion, use [GitHub Discussions](https://github.com/rakibulmehedi/ou_estimator/discussions).

Please search existing discussions before opening a new thread — your question may
already be answered.

## Bug Reports

If you've found a **reproducible bug**, open an
[Issue](https://github.com/rakibulmehedi/ou_estimator/issues/new?template=bug_report.yml)
using the bug report template. Include:

- App version (shown in the About screen)
- Device / OS version
- Steps to reproduce
- A small sample of input data if the bug is estimation-related

## Feature Requests

Use the
[Feature Request template](https://github.com/rakibulmehedi/ou_estimator/issues/new?template=feature_request.yml)
to suggest improvements. Describe the trading or analytical use case — not just the
implementation — so the request can be properly prioritized.

## Security Issues

Do **not** open a public issue for security vulnerabilities. See [SECURITY.md](SECURITY.md)
(when available) or email rakibulislammehedi4@gmail.com directly.
